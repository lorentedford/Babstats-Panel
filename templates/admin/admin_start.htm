Welcome to Neos.Chronos Administration
----------------------------------------------------------------------------

Here you will be able to use various features to administer your Chronos Stats, these features include:

* New Feature
* Delete players/squads,
* Edit players/squads,
* Delete maps/map stats,
* Upload images for maps,
* Reset weapon stats,
* Add/edit/delete servers,
* Add/edit/delete ranks,
* Add/edit/delete awards,
* Reset/repair/prune database,
* Database backup feature,
   (make sure backup folder permissions are set to 777)
* Merge players,
* Plus more....


For help with the usage of any features in this section go to the support forums at www.babstats.com
